# 0xSlots

**Collectively Owned Slots Protocol**

0xSlots is a protocol for creating **Harberger-taxed slots** on Ethereum. Each slot is a standalone smart contract where the occupant self-assesses a price, pays continuous tax to the slot's recipient, and can be force-bought by anyone willing to pay that price.

## Partial Common Ownership

1. **Self-assessed pricing** — The occupant sets their own price
2. **Continuous taxation** — Tax on the self-assessed price streams to the recipient
3. **Force-buy** — Anyone can buy the slot at the stated price

Set the price too high → excessive tax. Too low → someone buys it.
