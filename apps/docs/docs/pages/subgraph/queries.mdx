# Example Queries

Example GraphQL queries for the 0xSlots subgraph. You can run these against the [subgraph endpoints](/subgraph/overview#endpoints) or use the `SlotsClient` SDK methods.

## List Slots

```graphql
query GetSlots($first: Int!, $skip: Int) {
  slots(first: $first, skip: $skip, orderBy: createdAt, orderDirection: desc) {
    id
    recipient
    occupant
    price
    deposit
    taxPercentage
    collectedTax
    totalCollected
    currency {
      symbol
      decimals
    }
    module {
      name
      verified
    }
    createdAt
  }
}
```

**SDK equivalent:**

```ts
const { slots } = await client.getSlots({
  first: 10,
  orderBy: "createdAt",
  orderDirection: "desc",
});
```

## Slot Activity (All Events)

```graphql
query GetSlotActivity($slot: String!) {
  boughtEvents(where: { slot: $slot }, orderBy: timestamp, orderDirection: desc) {
    buyer
    previousOccupant
    price
    selfAssessedPrice
    timestamp
  }
  releasedEvents(where: { slot: $slot }, orderBy: timestamp, orderDirection: desc) {
    occupant
    refund
    timestamp
  }
  liquidatedEvents(where: { slot: $slot }, orderBy: timestamp, orderDirection: desc) {
    liquidator
    occupant
    bounty
    timestamp
  }
  settledEvents(where: { slot: $slot }, orderBy: timestamp, orderDirection: desc) {
    taxOwed
    taxPaid
    depositRemaining
    timestamp
  }
  taxCollectedEvents(where: { slot: $slot }, orderBy: timestamp, orderDirection: desc) {
    recipient
    amount
    timestamp
  }
  priceUpdatedEvents(where: { slot: $slot }, orderBy: timestamp, orderDirection: desc) {
    oldPrice
    newPrice
    timestamp
  }
  depositedEvents(where: { slot: $slot }, orderBy: timestamp, orderDirection: desc) {
    depositor
    amount
    timestamp
  }
  withdrawnEvents(where: { slot: $slot }, orderBy: timestamp, orderDirection: desc) {
    occupant
    amount
    timestamp
  }
  metadataUpdatedEvents(where: { slot: $slot }, orderBy: timestamp, orderDirection: desc) {
    author { id }
    uri
    rawJson
    adType
    timestamp
  }
}
```

**SDK equivalent:**

```ts
const activity = await client.getSlotActivity({ slot: "0x..." });
```
