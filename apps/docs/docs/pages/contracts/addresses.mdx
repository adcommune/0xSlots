## Programmatic Access

Use the `@0xslots/contracts` package to access addresses:

```ts
import {
  slotFactoryAddress,
  batchCollectorAddress,
  erc721SlotsAddress,
  getSlotsHubAddress,
  isSlotsHubDeployed,
  getSupportedChainIds,
} from "@0xslots/contracts";

// Address maps (chainId → address)
slotFactoryAddress[8453];    // "0xbf2F890E8F5CCCB3A1D7c5030dBC1843B9E36B0e"
slotFactoryAddress[84532];   // "0xc44De86e2A5f0C47f1Ba87C36DaBf54275814DEb"

// Helpers
const addr = getSlotsHubAddress(8453);        // Address | undefined
const deployed = isSlotsHubDeployed(8453);    // true
const chains = getSupportedChainIds();        // [8453, 84532]
```
