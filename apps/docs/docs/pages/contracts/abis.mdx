# ABIs

The `@0xslots/contracts` package exports all contract ABIs as typed viem-compatible ABI arrays.

## Installation

:::code-group

```bash [pnpm]
pnpm add @0xslots/contracts
```

```bash [npm]
npm install @0xslots/contracts
```

:::

## Available ABIs

```ts
import {
  slotAbi,            // Individual slot contract
  slotFactoryAbi,     // Factory / Hub contract
  batchCollectorAbi,  // BatchCollector utility
  erc721SlotsAbi,     // ERC721Slots (Harberger NFT collections)
  metadataModuleAbi,  // MetadataModule contract
} from "@0xslots/contracts";
```

## Usage with viem

```ts
import { createPublicClient, http } from "viem";
import { base } from "viem/chains";
import { slotAbi, slotFactoryAbi } from "@0xslots/contracts";

const client = createPublicClient({ chain: base, transport: http() });

// Read from a slot contract
const info = await client.readContract({
  address: "0x...",
  abi: slotAbi,
  functionName: "getSlotInfo",
});

// Read from the factory
const slotCount = await client.readContract({
  address: "0xbf2F890E8F5CCCB3A1D7c5030dBC1843B9E36B0e",
  abi: slotFactoryAbi,
  functionName: "slotCount",
});
```

## Usage with wagmi

```ts
import { useReadContract } from "wagmi";
import { slotAbi } from "@0xslots/contracts";

function SlotInfo({ address }: { address: `0x${string}` }) {
  const { data } = useReadContract({
    address,
    abi: slotAbi,
    functionName: "getSlotInfo",
  });
  // ...
}
```

## Key Functions

### `slotAbi`

| Function | Type | Description |
|---|---|---|
| `getSlotInfo` | Read | Full slot state in a single call |
| `buy` | Write | Buy or force-buy a slot |
| `selfAssess` | Write | Update self-assessed price |
| `topUp` | Write | Add to deposit |
| `withdraw` | Write | Withdraw from deposit |
| `release` | Write | Release the slot |
| `collect` | Write | Collect accumulated tax |
| `liquidate` | Write | Liquidate an insolvent slot |
| `proposeTaxUpdate` | Write | Propose tax rate change |
| `proposeModuleUpdate` | Write | Propose module change |
| `cancelPendingUpdates` | Write | Cancel pending updates |
| `setLiquidationBounty` | Write | Set liquidation bounty |
| `multicall` | Write | Batch multiple calls |

### `slotFactoryAbi`

| Function | Type | Description |
|---|---|---|
| `createSlot` | Write | Deploy a new slot |
| `createSlots` | Write | Deploy multiple slots |
| `slotCount` | Read | Total deployed slots |

### `metadataModuleAbi`

| Function | Type | Description |
|---|---|---|
| `updateMetadata` | Write | Set metadata URI for a slot |
| `tokenURI` | Read | Read metadata URI for a slot |
| `name` | Read | Module name ("MetadataModule") |
